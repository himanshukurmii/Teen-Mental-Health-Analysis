# Teen Mental Health Analysis Project

## Project Overview

This project focuses on analyzing the impact of social media usage on teen mental health using a real-world dataset. The main goal is to understand how daily social media usage affects stress levels, anxiety, addiction, sleep patterns, academic performance, and depression risk among teenagers.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model for Depression Prediction
- GitHub Project Documentation

---

## Problem Statement

Teenagers spend a significant amount of time on social media platforms, which may negatively affect their mental health and daily lifestyle. This project aims to identify patterns and relationships between social media usage and mental health indicators such as stress, anxiety, addiction, sleep quality, and depression.

The objective is to use data analytics and machine learning to predict depression risk and generate meaningful insights.

## Key Insights

### 1. Higher social media usage increases addiction levels

Students spending more hours on social media showed higher addiction scores.

### 2. Less sleep leads to higher stress and anxiety

Poor sleep quality strongly affects emotional and academic performance.

### 3. Physical activity improves mental wellness

Students with better physical activity levels had lower stress.

### 4. Depression risk is strongly linked with low sleep, high stress, and high addiction

These factors showed the strongest relationship with depression.

---

## Final Conclusion

Teen mental health is strongly influenced by digital lifestyle habits. Excessive social media use, especially before sleep, negatively affects sleep quality, stress, anxiety, and academic performance.

Balanced screen time, proper sleep, and regular physical activity can significantly improve mental health among teenagers.

This project helps students, parents, schools, and researchers understand the importance of healthy digital habits.
